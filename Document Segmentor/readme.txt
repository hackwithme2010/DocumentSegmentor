All the necessary paths for BBC dataset has been pre-set. 

You don't need to modify anything in the code. 

The output folder can be of any name, you have to keep an eye on the output of the particular ipynb notebook cell if you want to know the specific name. 

The output folder will be of name: sorted_(some_number) in your main directory.

Python libraries needed:
numpy
pandas
glob
shutil
sklearn

If possible, try to run it in Amazon SageMaker jupyter notebook instance as it will provide guaranteed results. 

This has not been tested in any other IDE.
